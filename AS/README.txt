Abstração: O primeiro conceito de Programação Orientada a Objetos presente no projeto desenvolvido é o da abstração de um sistema que gerencia itens de uma biblioteca a partir da criação de classes que representam esses objetos, como a classe Biblioteca, a classe Livro e a clsse ItemBiblioteca.

Encapsulamento:Ao longo do projeto, estão presentes outros conceitos de Programação Orientada a Objetos, como encapsulamento. Ao criar uma propriedade com o modificador de acesso protected chamada DescricaoInterna, ocorre então o encapsulamento do cógido protegendo esses dados de serem corrompidos ou modificados de maneira incorreta.

Herança:O conceito de herança foi utilizado ao criar a classe Livro que HERDA os atributos da classe abstrata ItemBiblioteca que serve como classe base nesse projeto.

Polimorfismo: Por fim, o conceito de poliformismo se faz presente na classe Biblioteca ao utilizar métodos com sobrecarga, ou seja, nomes iguais porém assinaturas diferentes.
